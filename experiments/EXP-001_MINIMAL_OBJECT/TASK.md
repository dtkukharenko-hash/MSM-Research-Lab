# EXP-001 — Minimal Object

## Цель

Определить минимальный объект модели MSM.

Кандидаты:

- площадка;
- переход;
- ступень;
- лестница.

## Главный вопрос

Что можно определить наиболее устойчиво и с минимальным числом противоречий?

## Задача Claude

Провести критический аудит кандидатов.

Для каждого кандидата:

1. дать короткое рабочее определение;
2. указать, что невозможно определить без будущего;
3. указать, какие случаи ломают определение;
4. предложить способ проверки на ADA 2023.

## Запрещено

- искать точки входа;
- оценивать прибыль;
- предлагать стратегию;
- использовать красивый визуальный пример как доказательство.

## Ожидаемый результат

Файл REPORT.md с выводом:

- какой объект лучше проверять первым;
- какие определения нужно уточнить;
- какие гипотезы временно отложить.
