# EXP-004 — March Features

## Статус

ACTIVE / READY

## Главный вопрос

Как начинается сильное направленное движение?

## Гипотеза

H-MARCH-001:
Большинство сильных направленных движений начинается с большой направленной свечи, которая заметно
отличается от предыдущих свечей.

## Данные

- Актив: ADA/USDT.
- Период: 2023.
- Основной ТФ: 4H.
- Источник: Irobot/backtester только read-only.

## Метод

1. Не искать прибыль.
2. Не искать входы.
3. Не использовать ZigZag как доказательство.
4. Не использовать слово "марш" при выборе участков.
5. Не пытаться сразу автоматически найти все марши.
6. Взять 50 крупных направленных движений ADA 2023 4H.
7. Критерий крупного движения пока максимально простой:
   - несколько последовательных свечей одного направления;
   - или движение, которое визуально человек назвал бы сильным.
8. Отбор НЕ должен зависеть от существования марша.
9. После отбора проверить H-MARCH-001: есть ли у большинства этих движений общие признаки начала.
10. Найти кандидаты на направленные движения вручную/полуавтоматически.
11. Для каждого кандидата проверить:
   - есть ли первая сильная свеча;
   - насколько её тело больше среднего тела предыдущих 5–10 свечей;
   - закрылась ли свеча близко к направлению движения;
   - было ли после неё продолжение хотя бы 2–3 свечи;
   - были ли контрпримеры, где сильное направленное движение началось без большой свечи.

## Важно

Это не стратегия и не сигнал. Это исследование признака.

## Ожидаемый результат

Создать:

```text
experiments/EXP-004_MARCH_FEATURES/REPORT.md
```

REPORT.md должен содержать:

1. Сколько участков проверено.
2. Сколько сильных направленных движений имели стартовую сильную свечу.
3. Сколько не имели.
4. Какие типы контрпримеров найдены.
5. Можно ли считать признак H-MARCH-001 рабочим наблюдением.
6. Вердикт: ACCEPT / REJECT / OPEN.

Если нужны таблицы или CSV, сохранять их в:

```text
experiments/EXP-004_MARCH_FEATURES/artifacts/
```
