# EXP-004 — March Features — REPORT

## Статус

REPORT_READY · Вердикт: **REJECT** для H-MARCH-001 в проверенном строгом виде.

## Цель

Проверить практическое наблюдение H-MARCH-001:

> Большинство сильных направленных движений начинается с большой направленной свечи, которая заметно
> отличается от предыдущих свечей.

Это не стратегия, не сигнал и не поиск входов. ZigZag не использовался как доказательство. Отбор участков не
зависел от существования марша.

## Данные

- Актив: ADA/USDT.
- ТФ: 4H.
- Запрошенный период: 2023.
- Фактически доступное окно в read-only источнике Irobot/backtester:
  **2023-07-01 00:00 — 2023-12-31 20:00**, 1104 бара.
- Источник: `/home/nnv/Irobot/logs/msm/features/ada4h_features_v1.csv`, только чтение.

## Метод

1. Взяты 4H-свечи ADA за доступную часть 2023.
2. Построены кандидаты сильных направленных движений без ZigZag:
   - окна длительностью 3–8 баров;
   - net move в одну сторону;
   - большинство свечей и close-to-close шагов в направлении движения.
3. Кандидаты отсортированы по `abs(net return) * sqrt(duration)`.
4. Выбраны 50 крупнейших непересекающихся движений с буфером 2 бара между участками.
5. Для стартовой свечи каждого движения проверено:
   - тело свечи направлено в сторону движения;
   - тело >= 1.5 среднего тела предыдущих 10 свечей;
   - закрытие близко к направлению движения:
     - UP: `close_position_in_range >= 0.70`;
     - DOWN: `close_position_in_range <= 0.30`;
   - после стартовой свечи минимум 2 из следующих 3 close-to-close шагов продолжают направление.

Стартовая сильная свеча считалась найденной только если выполнены все четыре условия.

## Результаты

Проверено участков: **50**.

| Метрика | Количество |
|---|---:|
| Имели стартовую сильную свечу по полному критерию | 12 |
| Не имели стартовую сильную свечу по полному критерию | 38 |
| Доля с полным признаком | 24% |
| Стартовая свеча направлена в сторону движения | 47 |
| Стартовая свеча >= 1.5 среднего тела предыдущих 10 свечей | 21 |
| Закрытие стартовой свечи близко к направлению | 34 |
| Продолжение 2 из следующих 3 свечей | 36 |

Главный разрыв: направленность стартовой свечи встречается часто, но размер стартовой свечи не является
общим свойством большинства выбранных сильных движений.

## Контрпримеры

Найдены типы контрпримеров:

1. **Старт не крупнее локального фона.** 29 из 50 движений начинались свечой, тело которой было меньше
   `1.5 * mean(body, previous 10)`.
2. **Закрытие не у края движения.** В 16 случаях стартовая свеча не закрывалась близко к направлению
   движения.
3. **Продолжение не сразу.** В 14 случаях после стартовой свечи не было 2 из 3 следующих close-to-close
   шагов в направлении движения.
4. **Редкий противоположный старт.** В 3 случаях тело стартовой свечи было противоположным или нейтральным
   относительно итогового направления выбранного движения.

## Ответы на вопросы TASK

1. Сколько участков проверено: **50**.
2. Сколько сильных направленных движений имели стартовую сильную свечу: **12**.
3. Сколько не имели: **38**.
4. Типы контрпримеров: некрупный старт, закрытие не у края, отложенное продолжение, редкий противоположный
   старт.
5. Можно ли считать H-MARCH-001 рабочим наблюдением: **нет в строгой формулировке "обычно начинается с
   большой направленной свечи"**. Более слабое наблюдение "стартовая свеча часто направлена в сторону
   будущего движения" остаётся открытым, но это уже другая гипотеза.
6. Вердикт: **REJECT**.

## Вердикт

# REJECT

H-MARCH-001 не подтверждена на 50 сильных направленных движениях ADA 4H за доступную часть 2023:
полный стартовый признак найден только у **12/50** движений. Большая стартовая свеча не является свойством
большинства движений в этой выборке.
