# MSM Model Decision Log

## MODEL DECISION 001

Дата:
2026-07-09

Статус:
ACCEPTED

Основание:
EXP-002_PLATFORM_EXISTENCE
EXP-003_MINIMAL_OBJECT_REVIEW

Суть решения:
После REJECT площадки как минимального объекта лаборатория не переходит автоматически к следующему
локализованному объекту. Вместо этого вводятся две конкурирующие исследовательские парадигмы:

Парадигма A:
минимальный элемент рынка — локализованный объект.
Примеры: площадка, переход, марш, импульс.

Парадигма B:
минимальный элемент рынка — состояние рынка.
Примеры: направление, режим, смена режима, фаза движения.

Решение:
Дальнейшие эксперименты должны явно указывать, к какой парадигме они относятся.

Что НЕ утверждается:

- локализованные объекты окончательно не отвергнуты;
- режимная модель не доказана;
- площадка может быть переоткрыта только с новым строгим определением и null-контролем.
