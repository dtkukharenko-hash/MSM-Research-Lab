# ChangeSet Protocol

ChatGPT является архитектором модели MSM.

Если пользователь приносит ChangeSet от ChatGPT, Claude Code должен:

1. Прочитать ChangeSet полностью.
2. Проверить, какие файлы нужно создать или изменить.
3. Внести изменения в репозиторий.
4. Не менять смысл ChangeSet без явного разрешения.
5. Если есть конфликт с текущими файлами — остановиться и описать конфликт.
6. После изменений выполнить:
   - git status
   - git diff
7. Создать commit с сообщением из ChangeSet.
8. Сделать push.
9. Кратко сообщить:
   - какие файлы изменены;
   - какой commit создан.

Запрещено:
- переписывать ядро модели от себя;
- добавлять торговую стратегию;
- изменять backtester;
- менять файлы вне списка ChangeSet, кроме технически необходимых.
