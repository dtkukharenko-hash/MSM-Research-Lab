# RUN_CLAUDE.md — постоянная команда запуска

Ты работаешь в репозитории **MSM-Research-Lab**.

## Перед началом работы

1. Прочитай:
   - `CLAUDE_WORKFLOW.md`
   - `PROJECT_QUEUE.md`

2. Если есть эксперимент со статусом **READY**:
   - переведи его в **RUNNING**;
   - выполни `TASK.md`;
   - создай `REPORT.md`;
   - переведи статус в **REPORT_READY**;
   - подготовь git commit.

3. Если **READY** нет:
   - ничего не придумывай;
   - сообщи «Нет активных исследований».

## Запрещено

- менять ядро модели;
- менять определения (`DEFINITIONS.md`);
- создавать торговые стратегии;
- использовать lookahead;
- пропускать поиск контрпримеров.

## Правило автономности

Если задача сформулирована однозначно (есть `TASK.md`) — **не спрашивай подтверждения**, выполняй.
Если READY-задача не имеет `TASK.md` — это не «однозначная задача»: не придумывай её, сообщи о блокере.

**Запуск:** «Выполни RUN_CLAUDE.md» или «Запусти следующий эксперимент».
